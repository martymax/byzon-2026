# Video assety

Sem nahrajte video k ročníku 2025:

    assets/video/byzon-2025.mp4

Web ho automaticky přehraje v sekci **Minulé ročníky → ročník 2025**
(odkazováno z `data/content.json` jako `local_video`).

Pozn.: Soubor nebylo možné stáhnout automaticky – proxy prostředí blokuje
přístup na `byzon.cz` ("denied by policy"), proto ho přidejte ručně.
